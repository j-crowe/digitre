Digitre: handwritten digit recognizer



Digitre is a small interactive web app using Machine learning to identify handwritten digits submited by the user.

You can read a more in-depth technical description of Digitre in a blog post I wrote about it in my Data Science blog.

1. Front end

Bootstrap-based open source template design... HTML canvas element and javascript to build the drawing board...

2. Pushing data to the back end

JQuery code to push base64-encoded images to Flask server-side code (Python web framework)...

3. Handwritten digit recognition

The handwritten digit recgnition is done by a convolutional neural network (CNN) Machine learning model implemented using the Python interface of Google's Tensorflow open source software library...

4. In the pipeline

My next step (as soon as I find the time...) will be to use user-submitted digits to train the model. The idea is to store user-submitted digits, along with the model output and user-inputed correct digit, and use the data to train and improve the model.
