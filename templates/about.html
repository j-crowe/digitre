Digitre: handwritten digit recognizer

Digitre is a small interactive web app using Machine learning to identify handwritten digits submited by the user.

You can read a more in-depth technical description of Digitre in a blog post I wrote about it in my Data Science blog.

Hey there, my name is Luis! If you want to learn more about me, check out my personal website.

I am really interested in Data Science and Machine Learning. Digitre is a little side project of mine, in which I developed an interactive handwritten digit recognizer. I wanted to use the power of Machine learning
