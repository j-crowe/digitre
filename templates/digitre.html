Handwritten digit recognizer

Digitre is a simple machine learning
digit recognizer app.

  • How it works

1. You draw a digit

Use the canvas to draw a digit from 0 to 9 and submit it for recognition.

2. Digitre "reads" it

Digitre uses Machine learning (neural network) to recognize the digit.

3. Digitre learns

The digit and its correct id are used to train and improve the model.
